# Pun Game - Match 3 Puzzle

A simple match-3 puzzle game similar to Pet Connect, built with Python and Pygame.

## How to Play
- Click on adjacent items to swap them.
- Create lines of 3 or more identical items to remove them and score points.
- Items will drop down and new ones will fill the grid.

## Requirements
- Python 3.x
- Pygame: Install with `pip install pygame`

## How to Run
1. Install Pygame: `pip install pygame`
2. Run the game: `python main.py`

## Running on GitHub
Clone the repository and run locally, or use GitHub Codespaces with Python and Pygame installed.
